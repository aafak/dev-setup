# Access your Ubuntu Desktop like Windows using RDP
You can access your Ubuntu Desktop like Windows using RDP (Remote Desktop Protocol) by setting up an RDP server on Ubuntu and connecting from a Windows machine.

# Step 1: Install xRDP on Ubuntu
Run the following commands on your Ubuntu machine to install the RDP server:
```
sudo apt update
sudo apt install xrdp -y
```

# After installation, start and enable xrdp:
```
sudo systemctl enable --now xrdp
Step 2: Allow RDP Port (3389) in Firewall
```

# If you have UFW (Uncomplicated Firewall) enabled, allow RDP traffic:
```
sudo ufw allow 3389/tcp
sudo ufw reload
```

# Check the firewall status:
```
sudo ufw status
```

# Step 3: Configure xRDP to Use Your Desktop Environment
By default, xRDP might use an Xorg session, which may not be what you want. To force it to use your Ubuntu Desktop (GNOME, XFCE, etc.), follow these steps:

# For GNOME (default Ubuntu desktop):

echo "gnome-session" > ~/.xsession

# For XFCE (lightweight desktop, recommended for better performance): Install XFCE if not installed:
```
sudo apt install xfce4 xfce4-goodies -y
```

# Then configure it:
```
echo "startxfce4" > ~/.xsession
```
# Restart xrdp for changes to take effect:

```
sudo systemctl restart xrdp
```

# Step 4: Connect from Windows
On your Windows machine, open Remote Desktop Connection (mstsc).
Enter your Ubuntu IP address (e.g., 192.168.1.100).
Enter your Ubuntu username and password when prompted.
Click OK to start the remote session.  You should see following
